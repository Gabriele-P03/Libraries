\doxysubsection{Detailed Description}
An Illegal\+State\+Exception is thrown whenever any function is called at a bad time, i.\+e. wheter an another function had to be called before the one which threw this exception

e.\+g.

The foo() function must be called after have set flag as true, then\+:

void foo()\{ if(!flag)\{ throw new Illegal\+State\+Exception() \} \}
